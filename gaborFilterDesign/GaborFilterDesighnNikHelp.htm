GaborFilterDesighnNik Help.

In order to successfully use this GUI please follow the simple instructions beneath:

1.       Define the filter, by changing the Mother Gabor parameters or, by loading a previously saved filter (you can view your filter in both time and frequency domains)

2.       Load an image

3.       Finally, apply the filter

4.       See the filtering result sing the "Show" and "Filtered Image" button combination.

Enjoy.
